Kid-Friendly App

Project Description

This is a personal project app designed to help kids manage their daily tasks and activities. The app includes a to-do list, money tracker, contacts, and settings. It is built using Android Studio, Kotlin, and Jetpack Compose.

Why I Built This

Many kids are given phones at an early age, but few apps are designed to balance fun with parental peace of mind. I wanted to create a digital environment where children can complete tasks, track money, and communicateâ€”without distractions or unsafe content.

Challenges Faced & Lessons Learned

One challenge was implementing persistent data saving across screens using Jetpack Compose. I learned how to manage state effectively, structure composables cleanly, and improve app performance.
