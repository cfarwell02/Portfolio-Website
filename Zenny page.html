Zenny

Project Description

Zenny is a smart, supportive money-tracking app designed to make personal finance feel simple, encouraging, and even fun. Snap a pic of your receipts, tag your expenses, and watch Zenny turn your spending habits into clear insights. With features like custom budgets, smart notifications, and savings goal tracking, Zenny helps you stay in control of your money—without the stress.

Why I Built This

Managing money can be stressful and confusing, especially for young adults. I wanted to create an app that makes tracking spending and saving approachable, positive, and even enjoyable—helping users build better habits without feeling overwhelmed.

Key Features

  • Snap and store receipts instantly
  • Tag expenses for easy organization
  • Visualize spending with clear insights
  • Custom budgets and smart notifications
  • Savings goal tracking
  • Friendly, stress-free user experience

Challenges & Lessons Learned

Building Zenny required integrating camera/photo features and real-time data sync—all while keeping the UI simple and welcoming. I learned a lot about React Native, Firebase, and designing for user engagement and accessibility.

  • Integrated React Native camera and image picker modules
  • Used Firebase Firestore for real-time data storage and sync
  • Implemented push notifications and custom reminders
  • Designed a friendly, accessible UI for all users

Technologies Used

  • React Native
  • Firebase (Firestore, Auth, Storage)
  • JavaScript
  • Expo (for rapid development and deployment)
  • Bootstrap Icons
  • Canva (for logo design)
