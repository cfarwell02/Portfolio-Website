Portfolio Website

Project Description

This is my personal portfolio website, built from the ground up to highlight my software development projects, blog posts, and capabilities. It showcases clean design, responsive layout, animations, and real-time data integration via Supabase—all without a custom server.

Why I Built This

I wanted to create a dedicated online space where I could display my skills, apps, and engineering journey. Beyond just a resume, this website serves as a living project—where I can experiment with front-end tools, improve my design sensibility, and build real functionality like live blog data and contact form handling.

Key Features

  • Modern, mobile-responsive design with Bootstrap
  • Animated sections using custom CSS and scroll effects
  • Dynamically loaded blog posts from Supabase
  • Contact form submissions saved securely to Supabase (no backend server required)
  • Project showcase with individual pages and GitHub/live links
  • Dedicated "About Me" and skills overview section

Challenges & Lessons Learned

The biggest technical hurdle was implementing real data persistence for my contact form using Supabase without relying on a custom backend. I also tackled responsive design, accessibility, and learned how to work with asynchronous JavaScript to fetch blog posts.

  • Learned how to use Supabase's JavaScript client to securely store contact form data
  • Improved my understanding of responsive design using Bootstrap’s grid system
  • Debugged CORS issues and learned how to test locally with live APIs
  • Gained experience working with async/await and form validation in Vanilla JS

This site is a reflection of where I am now—and where I’m headed as a developer.

Planned Improvements

Technologies Used
