Portfolio Website

Project Description

This is my personal portfolio website, built from scratch to highlight my software engineering projects, technical skills, and experience. It features a clean, modern design and responsive layout for seamless browsing across devices.

Why I Built This

As a developer, I needed a dedicated space to share my work, story, and skill set with potential employers and collaborators. Building my own site also gave me hands-on experience with front-end web technologies and best practices for accessibility, SEO, and user experience.

Website Preview

Features

  • Modern, mobile-responsive design
  • Project gallery with detailed pages
  • Contact form (with backend integration)
  • Links to GitHub and live projects
  • About and skills sections

Challenges Faced & Lessons Learned

My biggest challenge was connecting the contact form to a real backend (using Supabase) for form submissions, while keeping everything secure. I also learned how to design for accessibility and optimize the website for fast loading and smooth user experience.

Planned Improvements

  • Blog section for technical articles
  • Dark mode toggle
  • More interactive project demos
  • Expanded SEO and analytics

Technologies Used

  • HTML5 & CSS3
  • JavaScript (Vanilla)
  • Bootstrap 5
  • Supabase (for backend/contact form)
  • GitHub Pages (hosting)
